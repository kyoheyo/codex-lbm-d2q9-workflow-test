## 远端仓库准备

因为 GitHub connector 不能创建 repository，用户需要：

1. 在 GitHub 创建一个空仓库，不初始化 README、license 或 .gitignore。
2. 确保 Codex GitHub App 对该仓库有读写权限。
3. 向 Codex 提供 owner/repo。

实际仓库：https://github.com/kyoheyo/codex-lbm-d2q9-workflow-test

~~~text

~~~
